# 🌿 VerdantIQ — AI-Powered Digital Twin & IoT Intelligence Platform for Smart Greenhouses

**VerdantIQ** is a production-inspired Smart Greenhouse IoT Simulator that recreates a complete connected agriculture ecosystem entirely in software. The platform simulates real-world IoT sensors and actuators, enabling continuous environmental monitoring, autonomous greenhouse control, and AI-driven decision support without requiring physical hardware.

Designed around a digital twin architecture, VerdantIQ models temperature, humidity, soil moisture, CO₂ concentration, light intensity, and water levels while implementing intelligent automation through virtual irrigation systems, ventilation, lighting, and climate control. An event-driven rule engine dynamically maintains optimal growing conditions and adapts to changing environmental scenarios.

Beyond real-time monitoring, VerdantIQ transforms sensor telemetry into actionable insights using predictive analytics, historical trend analysis, environmental scoring, anomaly detection, crop-specific optimization, and comparative analytics such as Yesterday vs Today environmental intelligence. The platform also simulates MQTT-based IoT communication, sensor failures, multi-greenhouse management, and interactive digital twin visualization to emulate production-grade smart farming systems.

Built with a modern full-stack architecture, VerdantIQ showcases scalable API design, cloud deployment, responsive dashboards, secure authentication, real-time analytics, and AI-powered recommendations—bridging IoT, cloud computing, data engineering, and intelligent automation into a unified AgriTech platform.

## ✨ Key Highlights

*   **🌐 End-to-End IoT Architecture Simulation:** Simulated sensor readings and actuator switches that mimic true hardware behavior.
*   **🛰️ Digital Twin Greenhouse Visualization:** Interactive, animated dashboard schemas that show the real-time state of vents, heaters, lights, and irrigation pipes.
*   **🌡️ Real-Time Virtual Sensor Network:** Continuous data generation for temperature, humidity, soil moisture, light, CO₂, and water tank levels.
*   **⚙️ Intelligent Automation & Rule Engine:** Custom user-defined rules that trigger automated climate adjustments (e.g. turning on ventilation if humidity peaks).
*   **📡 MQTT Communication Simulation:** Pub/sub broker visualization showing real-time telemetry JSON feeds and control topics.
*   **🤖 AI-Based Crop Health & Irrigation Recommendations:** Virtual agricultural expert proposing optimized setpoints based on selected crops (Tomatoes, Lettuce, Strawberries).
*   **📈 Historical Analytics & Environmental Trend Comparison:** Data-rich overlays showing Today vs. Yesterday comparisons, anomaly detection, and growth stages.
*   **🌦️ Weather-Aware Greenhouse Simulation:** Environmental models reacting to changing external conditions (Sunny, Rainy, Storm, Heatwave).
*   **🚨 Predictive Alerts & Sensor Failure Simulation:** Inject system faults like sensor lockups, heater failure, or plumbing leaks to test system resilience.
*   **🏭 Multi-Greenhouse Monitoring Dashboard:** Switch views and manage multiple separate greenhouse units simultaneously.
*   **📊 Interactive Analytics & Performance Insights:** High-quality visual trends and diagnostic reports.

## 🚀 Getting Started

To run the application locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/HIYA-Banerjee/VerdantIQ.git
    cd VerdantIQ
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the development server:**
    ```bash
    npm run dev
    ```
